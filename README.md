# Smart-Dustbin-IoT
A smart dustbin that uses ultrasonic sensors and voice commands to open automatically, providing touch-free disposal. It also tracks fill levels and sends alerts when nearly full, ensuring efficient and hygienic waste management.

1. Alerts user when dustbin is full.
2. Alerts user when dustbin is wet for long period to avoid bad odour.
3. Opens dustbin when user says "OPEN DUSTBIN".
